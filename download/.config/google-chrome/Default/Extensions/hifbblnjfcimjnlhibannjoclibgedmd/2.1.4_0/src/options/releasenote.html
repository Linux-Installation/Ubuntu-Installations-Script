MasterPassword Release 2.1.4

Thank you for using MasterPassword for Chrome! The easiest way to handle site-unique password on the web. If you want to understand what this is all about, please see here.

If you find any bugs or missing features in this plugin, please report them on the issue tracker.

Use OS' key-vault

You can now store your masterkey in GNOME Keyring, KWallet, OSX Keychain or Windows Password Vault. You will need a separate helper app. Please refer to the wiki for further information.

Changes since v2.1.3

  • Fix issue, never autosubmit login form when multiple alternatives exists
  • Fix issue, invalid key id case mismatch warning on .mpsites import
